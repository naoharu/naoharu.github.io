How to install Beastmaker 2000 at home

結構前から質問が多かった、Beastmaker 2000　の取り付け方法！
最低限の情報を図にしたら下記の感じかな。 その他、よくある質問。

  • Beastmaker 2000って実際どうですか？
    • 負荷は高いけど確実に保持力は上がります！ただ保持に自身がない人だと1000シリーズでも十分楽しめると思います
  • 木の板とフィンガーボード本体はどうやって固定していますか？
    • 市販の木工ボンドで固定しています。
  • 木の板はどこで買いましたか？
    • ホームセンター（コーナン）です。
  • 木の板が乗っている鴨居の厚みはどのくらいですか？
    • 7〜8mm くらいです。
